Phoenix Tracing

You have more interaction with Phoenix Tracing. This User Interface allows you to see and search each important step in a query or insertion, all they way from the client through into the HBase side, and back again.

Apache Phoenix is sql query engine for Apache Hbase.

It helps managing and querying the Hbase tables. The web app is used to visualize the tracing information. It helps to predict and analyze query information and system matrices

The Apache Phoenix Tracing Web app has 5 features.

  1. List - lists the the traces with user preferred limits with access to it's dependency tree and timelines.
  2. Count Chart - Categorizing the traces according to the description.
  3. Trace Distribution - Visualizing the trace counts based on the host they are running.
  4. Trace Timeline - Visualizing the traces along the time axis.
  5. Dependency Tree - Creating the tree view of the traces.
